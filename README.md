# 🐍 Python Exercises

Repository kumpulan latihan Python dari dasar hingga visualisasi data.  
Latihan ini dikerjakan secara bertahap setiap hari untuk membangun fondasi pemrograman Python.

---

## 📁 Struktur Folder

### 📂 Belajar Python H-1 — Dasar-Dasar Python

Latihan pertama yang membahas konsep fundamental Python.

| File | Deskripsi |
|------|-----------|
| `tipe_data.py` | Pengenalan tipe data dasar: `str`, `int`, `float`, `bool` beserta 3 cara format output (`f-string`, `.format()`, `%`) |
| `tipe_data2.py` | Operasi **List**: menambah (`append`), mengubah (indexing), dan menghapus (`remove`) elemen dalam list hobi |
| `tipe_data3.py` | Perbandingan lengkap 5 tipe data kolektif: **List**, **Tuple**, **Set**, **Dictionary** dalam konteks data mahasiswa |
| `aritmatika.py` | Operasi aritmatika dan perbandingan: menghitung persentase penjualan dan logika bonus karyawan |
| `if_else.py` | Percabangan `if-elif-else`: menentukan status stok barang (melimpah / aman / perlu restock) |
| `absensi.py` | Sistem grading absensi sederhana menggunakan `if-elif-else` |
| `perulangan.py` | Perulangan `for` untuk menampilkan daftar harga barang dari list |

### 📂 Belajar Python H-2 — Struktur Data Lanjutan

| File | Deskripsi |
|------|-----------|
| `keranjang_belanja.py` | Implementasi **list of dictionaries** untuk membuat keranjang belanja berisi produk dan harga |

### 📂 Belajar Python H-3 — Visualisasi & Grafika

| File | Deskripsi |
|------|-----------|
| `mahasiswa.py` | Analisis data nilai 100 mahasiswa menggunakan **NumPy** & **Matplotlib**: statistik deskriptif, **3D Surface Plot**, **Histogram**, dan **Pie Chart** distribusi grade |
| `hati.py` | Animasi grafis bentuk hati menggunakan modul **Turtle** dengan efek warna gradient dan tulisan "I Love You" |
| `data.py` | *(File kosong — disiapkan untuk latihan selanjutnya)* |
| `visualisasi_nilai3D.png` | Output gambar hasil visualisasi dari `mahasiswa.py` |

## 🛠️ Teknologi

- **Python 3**
- **NumPy** — komputasi numerik
- **Matplotlib** — visualisasi data 2D & 3D
- **Turtle** — grafika bawaan Python

## 🚀 Cara Menjalankan

```bash
# Clone repository
git clone https://github.com/dimsdevv/python-exercises.git

# Masuk ke folder
cd python-exercises

# Jalankan file yang diinginkan
python "Belajar Python H-1/tipe_data.py"
python "Belajar Python H-3/mahasiswa.py"
```

> Untuk file `mahasiswa.py`, pastikan sudah install dependency:
> ```bash
> pip install numpy matplotlib
> ```
